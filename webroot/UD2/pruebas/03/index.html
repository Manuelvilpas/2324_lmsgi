Trabajando con java

Para realizar un documento en java

				
public class Holamundo {
	public static void main (String[] args) {
		System.out.println("Hola mundo");
	}
}

Puedes declarar una variable de texto String holaMundo que contenga el texto Hola mundo

Para compilar el archivo abrimos la consola y escribimos javac Holamundo.java

Para ejecutarlo habra que escribir en la misma consola java Holamundo.class y nos escribira en la consola Hola mundo
